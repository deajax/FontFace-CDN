# 使用方法

使用 `@font-face` 来引入字体链接

```
@font-face {
	font-family: "Flama";
	font-weight: normal;
	font-style: normal;
	src: url('https://cdn.jsdelivr.net/gh/deajax/FontFace-CDN/src/fonts/字体目录/字体名称.后缀') format('opentype');
}
```

> otf格式：format('opentype')
>
> ttf格式：format('truetype')
